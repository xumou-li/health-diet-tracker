FoodPage -> MealAPI : 请求食物列表

DB --> MealAPI : 返回食物列表
MealAPI --> FoodPage : 返回食物列表
U -> FoodPage : 选择食物并输入重量
FoodPage -> MealAPI : 提交新增记录
MealAPI -> DB : 校验参数与食物状态
MealAPI -> DB : 获取或创建身体快照
MealAPI -> DB : 按重量换算营养并保存记录
DB --> MealAPI : 返回保存结果
MealAPI --> FoodPage : 返回添加成功
FoodPage --> RecordPage : 返回并触发刷新
